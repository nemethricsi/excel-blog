A VLOOKUP (vagy magyar nyelven: FKERES) az egyik leggyakrabban használt Excel függvény. Ha gyakran dolgozol nagy adatkészletekkel, vagy gyakran kell keresni az Excel táblázataidban, ez a függvény a te barátod lesz.

Szintaxisa:

```excel
=FKERES(érték; táblázat; oszlop; [pontos]; [rendezett])
```

Some JS:

```js
const a = 'alma';
```

Some TS:

```ts
const a: string = 'alma';
```

```html
<p>alma</p>
```

```css
  color: red;
```
